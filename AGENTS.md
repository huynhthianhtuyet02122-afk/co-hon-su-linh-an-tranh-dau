# AGENTS

**Trước khi code, hãy đọc AGENTS.md trước. Chỉ đọc thêm file được task yêu cầu.**

## Vai trò

- ChatGPT: mentor/PM/game design/review repo.
- Codex: code Unity/C# theo task.
- RooCode: dọn tài liệu, Git, Gitleaks, kiểm tra an toàn, commit/push khi được xác nhận.

## Quy tắc

- Không tự ý tạo Unity project nếu task chưa yêu cầu.
- Không tự ý sửa concept gốc.
- Không tự ý commit/push.
- Không đọc toàn bộ concept dài nếu task không cần.

## Token & Output Rules

- Xem AGENT_TOKEN_RULES.md để biết quy tắc tối ưu input/output token.
- Không in full file, full diff, full log nếu người dùng không yêu cầu.
- Chỉ đọc concept dài khi task cần chi tiết game design.
