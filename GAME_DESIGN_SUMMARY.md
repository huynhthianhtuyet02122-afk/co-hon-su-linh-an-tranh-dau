# GAME DESIGN SUMMARY

**Cơ Hồn Sư: Linh Ấn Tranh Đấu** là game Unity 3D fantasy anime hiện đại, offline single-player.

## Core fantasy

- Người chơi là Cơ Hồn Sư.
- Gameplay xoay quanh Linh Cơ Đấu, bi-da ma thuật, Linh Cầu và Linh Ấn.
- Cơ Hồn Thẻ là menu gameplay và công cụ kể chuyện.
- Thiên Khung Hội là tổ chức đứng sau Linh Cơ Đấu.
- Elias Vân là phản diện tư tưởng.
- An Nhiên là nhân vật quan trọng đã biến mất.

## Luật game đã chốt

- Game là offline single-player.
- Trụ cột gameplay: bi-da ma thuật + RPG + leo rank + Đảo Lưu Đày.
- Mỗi rank có 5 sao.
- Thắng +1 sao.
- Thua có sao thì -1 sao.
- Thua ở 0 sao không mất sao, không rớt rank.
- Thua luôn vào Đảo Lưu Đày.
- Linh Cầu rơi vào lỗ phong ấn Linh Ấn.

## Scope hiện tại

- Chưa tạo Unity project.
- Chưa code gameplay.
- GAME_DESIGN_SUMMARY.md chỉ là bản tóm tắt design đã chốt để agent đọc nhanh.
- Concept dài nằm trong `docs/concept_co_hon_su_v4_chuyen_nghiep.md` và chỉ đọc khi task cần chi tiết game design.
