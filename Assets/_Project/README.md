# CoHonSu Official Unity Foundation v0.1

Foundation tối thiểu cho project Unity chính thức.

## Nội dung

- `Scripts/Core`: bootstrap và trạng thái runtime nền.
- `Scripts/Data`: cấu hình/profile placeholder.
- `Scripts/SaveLoad`: model save và save service local placeholder.
- `Scripts/Debugging`: debug panel runtime tối thiểu.

## Cách dùng nhanh

1. Tạo một GameObject trống trong scene.
2. Gắn `GameBootstrap`.
3. Bật `showDebugPanel` nếu cần xem trạng thái foundation khi Play Mode.

Chưa có gameplay bi-da, AI, online, gacha hoặc nội dung Đảo Lưu Đày.
